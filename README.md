# codeAlpha_HangmanGame
A console-based Hangman Game that challenges players to guess words letter by letter. 
Hangman Game (Console-Based)
Project Overview :

This project was developed as part of the CodeAlpha Python Programming Internship. It is a console-based Hangman game created using Python. The player guesses a hidden word one letter at a time within a limited number of attempts. The game provides feedback for correct and incorrect guesses and ends when the player either guesses the word or runs out of attempts.

Features :
Console-based interactive gameplay
Random word selection
Letter-by-letter guessing
Tracks correct and incorrect guesses
Limited number of attempts
Win and lose messages
Input validation for a better user experience

Technologies Used :
- Python 3
- PyCharm

Project Structure :
Hangman-Game/

How to Run :
1.Clone or download this repository.
2.Open the project in PyCharm or any Python IDE.
3.Ensure Python 3 is installed.
4.Open a terminal in the project folder and run:
    python main.py
5.Follow the on-screen instructions to play the game.

Gameplay :
- The computer randomly selects a word.
- Guess one letter at a time.
- Correct guesses reveal the letter in the word.
- Incorrect guesses reduce the remaining attempts.
- The game ends when you guess the word or run out of attempts.
